# 💎 GSE Premium Analytics Terminal

### Executive Overview
This terminal is a sophisticated data visualization tool designed to provide actionable insights into the **Ghana Stock Exchange (GSE)**. It bridges the gap between raw market data and investment intelligence, ideal for portfolio managers and risk analysts.

### 🛠️ Technical Stack
- **Frontend:** Streamlit (Web Interface)
- **Data Visualization:** Plotly (Interactive Charts)
- **Processing:** Python (Pandas, Requests)
- **Architecture:** Implements `lru_cache` for optimized data fetching and performance.

### ✨ Key Features
- **Market Dashboard:** Real-time overview of GSE equity performance and market trends.
- **Dynamic Charting:** Interactive price-action analysis for major tickers.
- **Risk Assessment:** Built with a credit risk perspective to monitor volatility and stock health.
